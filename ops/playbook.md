## Diagnosis Framework
When metrics underperform:
- Low traffic → HOOK problem → rewrite headlines, add pSEO pages, improve meta descriptions
- Traffic but low signups → STORY problem → improve landing page copy, add social proof, improve demo
- Signups but no upgrades → OFFER problem → improve free-to-paid trigger, add value to Pro, adjust pricing
- High churn → PRODUCT problem → fix bugs, add features users request, improve onboarding
